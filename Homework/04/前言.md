## Welcome

Welcome to **"讓 AI 教你編譯器 -- 理論與實作（LLVM 版）"** (Learn Compilers with AI -- Theory and Implementation, LLVM Edition).

This textbook represents a unique approach to learning compiler construction. Rather than presenting compiler theory in isolation, we integrate it with hands-on experience using **LLVM**, the industry-standard compiler framework that powers some of the world's most important software systems.

Whether you are a university student taking your first compiler course, a self-taught programmer curious about how languages work, or a professional engineer seeking to deepen your understanding of compilation technology, this book will guide you through the fascinating journey from source code to machine code.

## Getting Started

### Installation Guide

#### Ubuntu / Debian

```bash
# Add LLVM repository
wget https://apt.llvm.org/llvm.sh
chmod +x llvm.sh
sudo ./llvm.sh 14

# Install LLVM and Clang
sudo apt-get update
sudo apt-get install llvm-14 llvm-14-dev clang-14

# Verify installation
llvm-config --version
clang --version
```

#### macOS

```bash
# Using Homebrew
brew install llvm

# Add to PATH (add to ~/.zshrc or ~/.bashrc)
echo 'export PATH="/usr/local/opt/llvm/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

#### Windows

```powershell
# Download from https://releases.llvm.org/
# Or use Chocolatey
choco install llvm

# Verify installation
llvm-config --version
```

### Verify Your Setup

```bash
# Check LLVM version
llvm-config --version
# Expected: LLVM version 14.0.0 or higher

# Check Clang version
clang --version
# Expected: clang version 14.0.0 or higher

# Check llc (LLVM compiler)
llc --version
# Expected: Shows available targets

# Check opt (optimizer)
opt --version
```

### Quick Test

Create and run your first LLVM IR program:

```llvm
; hello.ll
define i32 @main() {
    ret i32 0
}
```

```bash
# Compile and run
llvm-as hello.ll -o hello.bc
llc hello.ll -o hello.s
clang hello.s -o hello
./hello
echo $?  # Should print 0
```

## Why Learn Compilers?

Compilers are among the most fascinating and complex software systems ever created. They serve as the bridge between human-readable source code and machine-executable instructions. Every time you write a program in C, C++, Rust, Swift, or any other compiled language, a compiler transforms your high-level intentions into precise machine operations.

Understanding compilers is not merely an academic exercise. It provides:

1. **Deeper programming insight**: Knowing how your code is translated helps you write more efficient programs
2. **Better debugging skills**: Understanding error messages and assembly output becomes intuitive
3. **Career opportunities**: Compiler engineering is a specialized field with high demand
4. **Foundation for innovation**: Many domain-specific languages and tools require compiler knowledge

## The LLVM Revolution

Traditional compiler courses often focused on theoretical concepts without providing hands-on experience with production-grade tools. The LLVM project changed this paradigm entirely.

LLVM (Low Level Virtual Machine) is:
- **Modular**: Each phase of compilation is a separate library
- **Widely adopted**: Used in Swift, Rust, Clang, Julia, and many other languages
- **Well-documented**: Extensive documentation and an active community
- **Research-friendly**: Easy to extend and experiment with

By learning compiler theory through LLVM, you gain practical skills that are directly applicable in industry while mastering fundamental computer science concepts.

## Book Structure

The book is organized into seven chapters, following the natural stages of compilation:

```
Source Code → Lexical Analysis → Syntax Analysis → Semantic Analysis 
           → IR Generation → Optimization → Code Generation → Executable
```

Each chapter follows a consistent structure:
1. **Conceptual introduction** of the topic
2. **Detailed explanation** with diagrams where helpful
3. **Code examples** demonstrating implementation
4. **Summary** of key takeaways
5. **Exercises** for practice and deeper learning

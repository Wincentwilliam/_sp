## 3.1 Introduction

Lexical analysis (scanning) and syntax analysis (parsing) form the foundation of any compiler's frontend. These two phases transform raw source text into a structured representation that subsequent phases can process.

### 3.1.1 The Role of Frontend

```
┌────────────────────────────────────────────────────────────────────────┐
│                    FRONTEND ARCHITECTURE                               │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│   SOURCE CODE                                                         │
│       │                                                               │
│       ▼                                                               │
│   ┌──────────────┐                                                    │
│   │   LEXER      │    Tokenizes source text                           │
│   │  (Scanner)   │    • Groups characters into tokens                 │
│   │              │    • Removes whitespace/comments                   │
│   └──────┬───────┘    • Handles escape sequences                      │
│          │             • Reports invalid characters                    │
│          ▼                                                             │
│   ┌──────────────┐                                                    │
│   │   PARSER     │    Builds syntax tree                              │
│   │              │    • Validates token sequence                       │
│   │  Token Stream│    • Enforces grammar rules                         │
│   └──────┬───────┘    • Constructs AST                                │
│          │             • Reports syntax errors                        │
│          ▼                                                             │
│   ┌──────────────┐                                                    │
│   │     AST      │    Abstract Syntax Tree                            │
│   │              │    • Hierarchical program representation            │
│   │              │    • Ready for semantic analysis                    │
│   └──────────────┘                                                    │
│                                                                        │
└────────────────────────────────────────────────────────────────────────┘
```

### 3.1.2 Why Separate Lexer and Parser?

| Aspect | Single Phase | Two Phases | Benefit |
|--------|-------------|-----------|---------|
| **Modularity** | ❌ Mixed concerns | ✅ Separated | Easier maintenance |
| **Reuse** | ❌ Language-specific | ✅ Token definitions reusable | Multiple parsers |
| **Complexity** | ❌ Complex | ✅ Simpler | Easier implementation |
| **Performance** | ❌ One pass | ✅ Streaming | Better memory usage |
| **Error Messages** | ❌ Vague | ✅ Precise | Better diagnostics |

### 3.2.2 Complete Lexer Implementation

```cpp
// lexer.h
// A complete lexer for a C-like language

#ifndef LEXER_H
#define LEXER_H

#include <string>
#include <vector>
#include <unordered_map>
#include <cctype>
#include <stdexcept>

class Lexer {
public:
    // Token type enumeration
    enum class TokenType {
        // Keywords
        KEYWORD_IF,
        KEYWORD_ELSE,
        KEYWORD_WHILE,
        KEYWORD_FOR,
        KEYWORD_DO,
        KEYWORD_RETURN,
        KEYWORD_BREAK,
        KEYWORD_CONTINUE,
        KEYWORD_INT,
        KEYWORD_FLOAT,
        KEYWORD_DOUBLE,
        KEYWORD_CHAR,
        KEYWORD_VOID,
        KEYWORD_CONST,
        KEYWORD_STATIC,
        KEYWORD_STRUCT,
        
        // Literals
        INTEGER_LITERAL,
        FLOAT_LITERAL,
        STRING_LITERAL,
        CHARACTER_LITERAL,
        IDENTIFIER,
        
        // Operators
        OP_PLUS,          // +
        OP_MINUS,         // -
        OP_STAR,          // *
        OP_SLASH,         // /
        OP_PERCENT,       // %
        OP_ASSIGN,        // =
        OP_EQ,            // ==
        OP_NE,            // !=
        OP_LT,            // <
        OP_GT,            // >
        OP_LE,            // <=
        OP_GE,            // >=
        OP_AND,           // &&
        OP_OR,            // ||
        OP_NOT,           // !
        OP_INCREMENT,     // ++
        OP_DECREMENT,     // --
        OP_ADD_ASSIGN,    // +=
        OP_SUB_ASSIGN,    // -=
        OP_MUL_ASSIGN,    // *=
        OP_DIV_ASSIGN,    // /=
        
        // Punctuation
        PUNCT_LPAREN,     // (
        PUNCT_RPAREN,     // )
        PUNCT_LBRACE,     // {
        PUNCT_RBRACE,     // }
        PUNCT_LBRACKET,   // [
        PUNCT_RBRACKET,   // ]
        PUNCT_SEMICOLON,  // ;
        PUNCT_COMMA,      // ,
        PUNCT_COLON,      // :
        PUNCT_DOT,        // .
        PUNCT_ARROW,      // ->
        
        // Special
        TOKEN_EOF,
        TOKEN_ERROR,
        TOKEN_UNKNOWN
    };
    
    // Token structure
    struct Token {
        TokenType type;
        std::string lexeme;      // The actual text
        int line;                // Line number
        int column;              // Column number
        int intValue;            // For integer literals
        double floatValue;       // For float literals
        
        Token(TokenType t, const std::string& l, int ln, int col)
            : type(t), lexeme(l), line(ln), column(col), 
              intValue(0), floatValue(0.0) {}
    };
    
    explicit Lexer(const std::string& source);
    std::vector<Token> tokenize();
    const std::string& getSource() const { return source_; }
    
private:
    std::string source_;
    size_t position_;
    int line_;
    int column_;
    std::unordered_map<std::string, TokenType> keywords_;
    
    // Keyword initialization
    void initKeywords();
    
    // Character operations
    char currentChar();
    char peekChar(int offset = 1);
    void advance();
    bool isAtEnd();
    void skipWhitespace();
    void skipComment();
    
    // Token recognition
    Token makeToken(TokenType type, const std::string& lexeme);
    Token makeError(const std::string& message);
    Token scanIdentifier();
    Token scanNumber();
    Token scanStringLiteral();
    Token scanCharLiteral();
    Token scanOperatorOrPunctuation();
};

#endif // LEXER_H
```

```cpp
// lexer.cpp
// Implementation of the lexer

#include "lexer.h"
#include <iostream>
#include <sstream>

Lexer::Lexer(const std::string& source) 
    : source_(source), position_(0), line_(1), column_(1) {
    initKeywords();
}

void Lexer::initKeywords() {
    keywords_ = {
        // Control flow
        {"if", TokenType::KEYWORD_IF},
        {"else", TokenType::KEYWORD_ELSE},
        {"while", TokenType::KEYWORD_WHILE},
        {"for", TokenType::KEYWORD_FOR},
        {"do", TokenType::KEYWORD_DO},
        {"return", TokenType::KEYWORD_RETURN},
        {"break", TokenType::KEYWORD_BREAK},
        {"continue", TokenType::KEYWORD_CONTINUE},
        
        // Types
        {"int", TokenType::KEYWORD_INT},
        {"float", TokenType::KEYWORD_FLOAT},
        {"double", TokenType::KEYWORD_DOUBLE},
        {"char", TokenType::KEYWORD_CHAR},
        {"void", TokenType::KEYWORD_VOID},
        
        // Storage class
        {"const", TokenType::KEYWORD_CONST},
        {"static", TokenType::KEYWORD_STATIC},
        {"struct", TokenType::KEYWORD_STRUCT},
    };
}

// ============================================================================
// Character Operations
// ============================================================================

char Lexer::currentChar() {
    if (isAtEnd()) return '\0';
    return source_[position_];
}

char Lexer::peekChar(int offset) {
    size_t peekPos = position_ + offset;
    if (peekPos >= source_.size()) return '\0';
    return source_[peekPos];
}

void Lexer::advance() {
    if (!isAtEnd()) {
        if (currentChar() == '\n') {
            line_++;
            column_ = 1;
        } else {
            column_++;
        }
        position_++;
    }
}

bool Lexer::isAtEnd() {
    return position_ >= source_.size();
}

// ============================================================================
// Whitespace and Comments
// ============================================================================

void Lexer::skipWhitespace() {
    while (!isAtEnd()) {
        char c = currentChar();
        if (c == ' ' || c == '\t' || c == '\r' || c == '\n') {
            advance();
        } else if (c == '/' && peekChar() == '/') {
            // Single-line comment
            while (!isAtEnd() && currentChar() != '\n') {
                advance();
            }
        } else if (c == '/' && peekChar() == '*') {
            // Multi-line comment
            advance(); advance();  // Skip /*
            while (!isAtEnd()) {
                if (currentChar() == '*' && peekChar() == '/') {
                    advance(); advance();  // Skip */
                    break;
                }
                advance();
            }
        } else {
            break;
        }
    }
}

// ============================================================================
// Token Recognition
// ============================================================================

std::vector<Lexer::Token> Lexer::tokenize() {
    std::vector<Token> tokens;
    
    while (!isAtEnd()) {
        skipWhitespace();
        if (isAtEnd()) break;
        
        int startLine = line_;
        int startColumn = column_;
        
        char c = currentChar();
        
        // Identifier or keyword
        if (std::isalpha(c) || c == '_') {
            Token token = scanIdentifier();
            token.line = startLine;
            token.column = startColumn;
            tokens.push_back(token);
            continue;
        }
        
        // Number
        if (std::isdigit(c)) {
            Token token = scanNumber();
            token.line = startLine;
            token.column = startColumn;
            tokens.push_back(token);
            continue;
        }
        
        // String literal
        if (c == '"') {
            Token token = scanStringLiteral();
            token.line = startLine;
            token.column = startColumn;
            tokens.push_back(token);
            continue;
        }
        
        // Character literal
        if (c == '\'') {
            Token token = scanCharLiteral();
            token.line = startLine;
            token.column = startColumn;
            tokens.push_back(token);
            continue;
        }
        
        // Operator or punctuation
        Token token = scanOperatorOrPunctuation();
        token.line = startLine;
        token.column = startColumn;
        tokens.push_back(token);
    }
    
    // Add EOF token
    tokens.push_back(makeToken(TokenType::TOKEN_EOF, ""));
    tokens.back().line = line_;
    tokens.back().column = column_;
    
    return tokens;
}

Lexer::Token Lexer::scanIdentifier() {
    int startLine = line_;
    int startColumn = column_;
    std::string lexeme;
    
    // Read identifier characters
    while (!isAtEnd() && (std::isalnum(currentChar()) || currentChar() == '_')) {
        lexeme += currentChar();
        advance();
    }
    
    // Check if it's a keyword
    auto it = keywords_.find(lexeme);
    if (it != keywords_.end()) {
        return Token(it->second, lexeme, startLine, startColumn);
    }
    
    // It's an identifier
    return Token(TokenType::IDENTIFIER, lexeme, startLine, startColumn);
}

Lexer::Token Lexer::scanNumber() {
    int startLine = line_;
    int startColumn = column_;
    std::string lexeme;
    bool isFloat = false;
    
    // Integer part
    while (!isAtEnd() && std::isdigit(currentChar())) {
        lexeme += currentChar();
        advance();
    }
    
    // Check for decimal point
    if (currentChar() == '.' && std::isdigit(peekChar())) {
        isFloat = true;
        lexeme += currentChar();
        advance();
        
        // Fractional part
        while (!isAtEnd() && std::isdigit(currentChar())) {
            lexeme += currentChar();
            advance();
        }
    }
    
    // Check for exponent (scientific notation)
    if (currentChar() == 'e' || currentChar() == 'E') {
        isFloat = true;
        lexeme += currentChar();
        advance();
        
        if (currentChar() == '+' || currentChar() == '-') {
            lexeme += currentChar();
            advance();
        }
        
        while (!isAtEnd() && std::isdigit(currentChar())) {
            lexeme += currentChar();
            advance();
        }
    }
    
    // Create token
    Token token(isFloat ? TokenType::FLOAT_LITERAL : TokenType::INTEGER_LITERAL,
                lexeme, startLine, startColumn);
    
    if (isFloat) {
        token.floatValue = std::stod(lexeme);
    } else {
        token.intValue = std::stoi(lexeme);
    }
    
    return token;
}

Lexer::Token Lexer::scanStringLiteral() {
    int startLine = line_;
    int startColumn = column_;
    std::string lexeme;
    advance();  // Skip opening quote
    
    while (!isAtEnd() && currentChar() != '"') {
        if (currentChar() == '\\') {
            // Escape sequence
            lexeme += currentChar();
            advance();
            if (!isAtEnd()) {
                lexeme += currentChar();
                advance();
            }
        } else if (currentChar() == '\n') {
            // Unterminated string (error)
            return makeError("Unterminated string literal");
        } else {
            lexeme += currentChar();
            advance();
        }
    }
    
    if (isAtEnd()) {
        return makeError("Unterminated string literal");
    }
    
    advance();  // Skip closing quote
    return Token(TokenType::STRING_LITERAL, lexeme, startLine, startColumn);
}

Lexer::Token Lexer::scanCharLiteral() {
    int startLine = line_;
    int startColumn = column_;
    std::string lexeme;
    advance();  // Skip opening quote
    
    if (currentChar() == '\\') {
        lexeme += currentChar();
        advance();
        if (!isAtEnd()) {
            lexeme += currentChar();
            advance();
        }
    } else if (!isAtEnd() && currentChar() != '\'') {
        lexeme += currentChar();
        advance();
    }
    
    if (currentChar() != '\'') {
        return makeError("Unterminated character literal");
    }
    
    advance();  // Skip closing quote
    return Token(TokenType::CHARACTER_LITERAL, lexeme, startLine, startColumn);
}

Lexer::Token Lexer::scanOperatorOrPunctuation() {
    int startLine = line_;
    int startColumn = column_;
    char c = currentChar();
    advance();
    
    switch (c) {
        // Single-character punctuation
        case '(': return Token(TokenType::PUNCT_LPAREN, "(", startLine, startColumn);
        case ')': return Token(TokenType::PUNCT_RPAREN, ")", startLine, startColumn);
        case '{': return Token(TokenType::PUNCT_LBRACE, "{", startLine, startColumn);
        case '}': return Token(TokenType::PUNCT_RBRACE, "}", startLine, startColumn);
        case '[': return Token(TokenType::PUNCT_LBRACKET, "[", startLine, startColumn);
        case ']': return Token(TokenType::PUNCT_RBRACKET, "]", startLine, startColumn);
        case ';': return Token(TokenType::PUNCT_SEMICOLON, ";", startLine, startColumn);
        case ',': return Token(TokenType::PUNCT_COMMA, ",", startLine, startColumn);
        case ':': return Token(TokenType::PUNCT_COLON, ":", startLine, startColumn);
        case '.': return Token(TokenType::PUNCT_DOT, ".", startLine, startColumn);
        
        // Two-character operators
        case '+':
            if (currentChar() == '+') { advance(); return Token(TokenType::OP_INCREMENT, "++", startLine, startColumn); }
            if (currentChar() == '=') { advance(); return Token(TokenType::OP_ADD_ASSIGN, "+=", startLine, startColumn); }
            return Token(TokenType::OP_PLUS, "+", startLine, startColumn);
            
        case '-':
            if (currentChar() == '-') { advance(); return Token(TokenType::OP_DECREMENT, "--", startLine, startColumn); }
            if (currentChar() == '=') { advance(); return Token(TokenType::OP_SUB_ASSIGN, "-=", startLine, startColumn); }
            if (currentChar() == '>') { advance(); return Token(TokenType::PUNCT_ARROW, "->", startLine, startColumn); }
            return Token(TokenType::OP_MINUS, "-", startLine, startColumn);
            
        case '*':
            if (currentChar() == '=') { advance(); return Token(TokenType::OP_MUL_ASSIGN, "*=", startLine, startColumn); }
            return Token(TokenType::OP_STAR, "*", startLine, startColumn);
            
        case '/':
            if (currentChar() == '=') { advance(); return Token(TokenType::OP_DIV_ASSIGN, "/=", startLine, startColumn); }
            return Token(TokenType::OP_SLASH, "/", startLine, startColumn);
            
        case '%': return Token(TokenType::OP_PERCENT, "%", startLine, startColumn);
        case '=': 
            if (currentChar() == '=') { advance(); return Token(TokenType::OP_EQ, "==", startLine, startColumn); }
            return Token(TokenType::OP_ASSIGN, "=", startLine, startColumn);
            
        case '!':
            if (currentChar() == '=') { advance(); return Token(TokenType::OP_NE, "!=", startLine, startColumn); }
            return Token(TokenType::OP_NOT, "!", startLine, startColumn);
            
        case '<':
            if (currentChar() == '=') { advance(); return Token(TokenType::OP_LE, "<=", startLine, startColumn); }
            return Token(TokenType::OP_LT, "<", startLine, startColumn);
            
        case '>':
            if (currentChar() == '=') { advance(); return Token(TokenType::OP_GE, ">=", startLine, startColumn); }
            return Token(TokenType::OP_GT, ">", startLine, startColumn);
            
        case '&':
            if (currentChar() == '&') { advance(); return Token(TokenType::OP_AND, "&&", startLine, startColumn); }
            return Token(TokenType::TOKEN_UNKNOWN, "&", startLine, startColumn);
            
        case '|':
            if (currentChar() == '|') { advance(); return Token(TokenType::OP_OR, "||", startLine, startColumn); }
            return Token(TokenType::TOKEN_UNKNOWN, "|", startLine, startColumn);
    }
    
    return Token(TokenType::TOKEN_UNKNOWN, std::string(1, c), startLine, startColumn);
}

Lexer::Token Lexer::makeToken(TokenType type, const std::string& lexeme) {
    return Token(type, lexeme, line_, column_);
}

Lexer::Token Lexer::makeError(const std::string& message) {
    std::cerr << "Lexer error at line " << line_ << ", column " << column_ 
              << ": " << message << std::endl;
    return Token(TokenType::TOKEN_ERROR, message, line_, column_);
}
```

## 3.3 Regular Expressions and Finite Automata

Regular expressions provide a formal way to specify token patterns, and finite automata provide the mechanism to recognize them.

### 3.3.1 Regular Expression Basics

```

│                  REGULAR EXPRESSION REFERENCE                           │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│  SYMBOLS:                                                             │
│  ┌────────────┬──────────────────────────────────────────────────┐    │
│  │   Symbol   │  Description                                    │    │
│  ├────────────┼──────────────────────────────────────────────────┤    │
│  │    c       │  Match the character 'c'                         │    │
│  │    .       │  Match any single character                      │    │
│  │    *       │  Match zero or more of the preceding             │    │
```

### 3.3.2 Token Patterns as Regular Expressions

```cpp
// Token pattern definitions using regex-like syntax

struct TokenPattern {
    std::string name;           // Token name
    std::string regex;          // Pattern description
    TokenType type;            // Corresponding token type
};

std::vector<TokenPattern> tokenPatterns = {
    // Whitespace and comments (ignored)
    {"WS", "[ \\t\\n\\r]+", TokenType::TOKEN_EOF},        // Skip
    {"COMMENT", "//.*|\\/\\*[\\s\\S]*?\\*\\/", TokenType::TOKEN_EOF},  // Skip
    
    // Keywords
    {"IF", "if", TokenType::KEYWORD_IF},
    {"ELSE", "else", TokenType::KEYWORD_ELSE},
    {"WHILE", "while", TokenType::KEYWORD_WHILE},
    {"RETURN", "return", TokenType::KEYWORD_RETURN},
    
    // Literals
    {"INT", "[0-9]+", TokenType::INTEGER_LITERAL},
    {"FLOAT", "[0-9]+\\.[0-9]+", TokenType::FLOAT_LITERAL},
    {"STRING", "\"([^\"\\\\]|\\\\.)*\"", TokenType::STRING_LITERAL},
    
    // Identifiers
    {"ID", "[a-zA-Z_][a-zA-Z0-9_]*", TokenType::IDENTIFIER},
    
    // Operators
    {"PLUS", "\\+", TokenType::OP_PLUS},
    {"MINUS", "-", TokenType::OP_MINUS},
    {"STAR", "\\*", TokenType::OP_STAR},
    {"SLASH", "/", TokenType::OP_SLASH},
    {"ASSIGN", "==?", TokenType::OP_ASSIGN},  // = or ==
};
```

## 3.4 Syntax Analysis (Parsing)

Parsing analyzes the token stream according to grammar rules and produces a parse tree or Abstract Syntax Tree (AST).

### 3.4.1 Context-Free Grammars

A **context-free grammar (CFG)** consists of:

| Component | Description | Example |
|-----------|-------------|---------|
| **Terminals** | Token types | `if`, `+`, `IDENTIFIER` |
| **Non-terminals** | Syntactic categories | `expression`, `statement` |
| **Production rules** | How non-terminals expand | `expr → expr + term` |
| **Start symbol** | Entry point | `program` |

### 3.4.2 EBNF Grammar for Our Language

```ebnf
// ===================================================================
// PROGRAM STRUCTURE
// ===================================================================

program        ::= declaration*

declaration    ::= var-declaration 
                | fun-declaration 
                | struct-declaration

// ===================================================================
// TYPE SPECIFIERS
// ===================================================================

type-specifier ::= VOID | CHAR | INT | FLOAT | DOUBLE
                 | IDENTIFIER  // For user-defined types (struct names)

// ===================================================================
// VARIABLE DECLARATIONS
// ===================================================================

var-declaration ::= type-specifier IDENTIFIER ( '[' INTEGER_LITERAL ']' )? ';'

// ===================================================================
// FUNCTION DECLARATIONS
// ===================================================================

fun-declaration ::= type-specifier IDENTIFIER '(' params ')' compound-stmt

params          ::= param-list | VOID
param-list      ::= param ( ',' param )*
param           ::= type-specifier IDENTIFIER

// ===================================================================
// STATEMENTS
// ===================================================================

compound-stmt   ::= '{' local-declarations* statement-list* '}'

local-declarations ::= var-declaration+

statement-list  ::= statement*

statement       ::= expression-stmt
                | compound-stmt
                | selection-stmt
                | iteration-stmt
                | jump-stmt

expression-stmt ::= expression? ';'

selection-stmt  ::= IF '(' expression ')' statement ( ELSE statement )?

iteration-stmt  ::= WHILE '(' expression ')' statement
                | FOR '(' expression? ';' expression? ';' expression? ')' statement

jump-stmt       ::= RETURN expression? ';'
                | BREAK ';'
                | CONTINUE ';'

// ===================================================================
// EXPRESSIONS
// ===================================================================

expression      ::= assign-expr

assign-expr     ::= lvalue '=' assign-expr
                | logical-or-expr

lvalue          ::= IDENTIFIER
                | IDENTIFIER '[' expression ']'

logical-or-expr ::= logical-and-expr ( '||' logical-and-expr )*

logical-and-expr ::= equality-expr ( '&&' equality-expr )*

equality-expr   ::= relational-expr ( ( '==' | '!=' ) relational-expr )*

relational-expr ::= additive-expr ( ( '<' | '>' | '<=' | '>=' ) additive-expr )*

additive-expr   ::= multiplicative-expr ( ( '+' | '-' ) multiplicative-expr )*

multiplicative-expr ::= unary-expr ( ( '*' | '/' | '%' ) unary-expr )*

unary-expr      ::= ( '+' | '-' | '!' ) unary-expr
                | postfix-expr

postfix-expr    ::= primary-expr ( '[' expression ']' | '(' args ')' | '.' IDENTIFIER )*

primary-expr    ::= INTEGER_LITERAL
                | FLOAT_LITERAL
                | STRING_LITERAL
                | IDENTIFIER
                | '(' expression ')'

args            ::= assign-expr ( ',' assign-expr )* | ε
```

## 3.5 Context-Free Grammars

### 3.5.1 Ambiguity and Precedence

Grammars can be ambiguous, meaning a string may have multiple parse trees:

```
┌────────────────────────────────────────────────────────────────────────┐
│                    DANGLING ELSE AMBIGUITY                              │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│  Grammar:                                                              │
│    stmt → IF '(' expr ')' stmt                                         │
│          | IF '(' expr ')' stmt ELSE stmt                              │
│          | OTHER                                                        │
│                                                                        │
│  Input:  if (a) if (b) x = 1; else x = 2;                            │
│                                                                        │
│  ───────────────────────────────────────────────────────────────────  │
│                                                                        │
│  Interpretation 1 (else binds to inner if):                             │
│                                                                        │
│          stmt                                                          │
│          │                                                             │
│       IF (a)                                                           │
│          │                                                             │
│          └──► stmt                                                     │
│                  │                                                     │
│               IF (b)                                                  │
│                  │                                                     │
│               ┌──┴──┐                                                 │
│               │     │                                                 │
│             x=1    ELSE x=2                                           │
│                                                                        │
│  ───────────────────────────────────────────────────────────────────  │
│                                                                        │
│  Interpretation 2 (else binds to outer if):                            │
│                                                                        │
│          stmt                                                          │
│          │                                                             │
│       IF (a)                                                           │
│          │                                                             │
│       ┌──┴──┐                                                         │
│       │     │                                                         │
│    IF (b)  ELSE x=2                                                  │
│       │                                                             │
│     x = 1                                                            │
│                                                                        │
│  ───────────────────────────────────────────────────────────────────  │
│                                                                        │
│  Standard Resolution: Shift-reduce parser prefers shift,                │
│  so ELSE binds to nearest IF.                                         │
│                                                                        │
└────────────────────────────────────────────────────────────────────────┘
```

### 3.5.2 Eliminating Ambiguity

```ebnf
// Ambiguous grammar (problematic):
statement ::= IF '(' expr ')' statement 
            | IF '(' expr ')' statement ELSE statement
            | OTHER

// Unambiguous grammar (resolved):
statement ::= matched-stmt | unmatched-stmt

matched-stmt ::= IF '(' expr ')' matched-stmt ELSE matched-stmt
              | OTHER

unmatched-stmt ::= IF '(' expr ')' statement
                 | IF '(' expr ')' matched-stmt ELSE unmatched-stmt
```

### 3.6.2 Parser Implementation

```cpp
// parser.cpp
// Implementation of recursive descent parser

#include "parser.h"
#include <iostream>
#include <stdexcept>
#include <cassert>

// ============================================================================
// PARSER CONSTRUCTION
// ============================================================================

Parser::Parser(const std::vector<Lexer::Token>& tokens)
    : tokens_(tokens), current_(0) {}

// ============================================================================
// HELPER METHODS
// ============================================================================

Lexer::Token& Parser::current() {
    return tokens_[current_];
}

Lexer::Token& Parser::peek(int offset) {
    size_t idx = current_ + offset;
    if (idx >= tokens_.size()) return tokens_.back();
    return tokens_[idx];
}

void Parser::advance() {
    if (!check(Lexer::TokenType::TOKEN_EOF)) {
        current_++;
    }
}

bool Parser::check(Lexer::TokenType type) {
    return current().type == type;
}

bool Parser::checkAny(const std::vector<Lexer::TokenType>& types) {
    for (auto type : types) {
        if (check(type)) return true;
    }
    return false;
}

bool Parser::match(Lexer::TokenType type) {
    if (check(type)) {
        advance();
        return true;
    }
    return false;
}

bool Parser::matchAny(const std::vector<Lexer::TokenType>& types) {
    for (auto type : types) {
        if (check(type)) {
            advance();
            return true;
        }
    }
    return false;
}

void Parser::error(const std::string& message) {
    std::cerr << "Parse error at line " << current().line 
              << ", column " << current().column 
              << ": " << message << std::endl;
    throw std::runtime_error("Parse error");
}

void Parser::errorAtCurrent(const std::string& message) {
    error(message);
}

void Parser::synchronize() {
    advance();
    
    while (!check(Lexer::TokenType::TOKEN_EOF)) {
        if (prev().type == Lexer::TokenType::PUNCT_SEMICOLON) return;
        
        switch (current().type) {
            case Lexer::TokenType::KEYWORD_IF:
            case Lexer::TokenType::KEYWORD_WHILE:
            case Lexer::TokenType::KEYWORD_FOR:
            case Lexer::TokenType::KEYWORD_RETURN:
            case Lexer::TokenType::KEYWORD_INT:
            case Lexer::TokenType::KEYWORD_FLOAT:
            case Lexer::TokenType::KEYWORD_VOID:
                return;
            default:
                advance();
        }
    }
}

// ============================================================================
// PARSING: PROGRAM STRUCTURE
// ============================================================================

std::unique_ptr<Program> Parser::parseProgram() {
    auto program = std::make_unique<Program>();
    
    while (!check(Lexer::TokenType::TOKEN_EOF)) {
        try {
            auto decl = parseDeclaration();
            if (decl) {
                program->declarations.push_back(std::move(decl));
            }
        } catch (...) {
            synchronize();
            if (!check(Lexer::TokenType::TOKEN_EOF)) {
                advance();
            }
        }
    }
    
    return program;
}

std::unique_ptr<Decl> Parser::parseDeclaration() {
    std::string type = parseTypeSpecifier();
    
    if (check(Lexer::TokenType::IDENTIFIER)) {
        std::string name = current().lexeme;
        advance();
        
        if (check(Lexer::TokenType::PUNCT_LPAREN)) {
            // Function declaration
            return parseFunctionDeclaration(type, name);
        } else {
            // Variable declaration
            return parseVarDeclaration(type, name);
        }
    }
    
    error("Expected identifier after type");
    return nullptr;
}

std::unique_ptr<VarDecl> Parser::parseVarDeclaration(const std::string& type, const std::string& name) {
    int arraySize = 0;
    
    if (match(Lexer::TokenType::PUNCT_LBRACKET)) {
        if (check(Lexer::TokenType::INTEGER_LITERAL)) {
            arraySize = current().intValue;
            advance();
        }
        match(Lexer::TokenType::PUNCT_RBRACKET);
    }
    
    match(Lexer::TokenType::PUNCT_SEMICOLON);
    
    return std::make_unique<VarDecl>(type, name, arraySize);
}

std::unique_ptr<FunctionDecl> Parser::parseFunctionDeclaration(const std::string& returnType, 
                                                               const std::string& name) {
    auto func = std::make_unique<FunctionDecl>(returnType, name);
    
    // Parse parameters
    match(Lexer::TokenType::PUNCT_LPAREN);
    
    if (!check(Lexer::TokenType::PUNCT_RPAREN)) {
        do {
            std::string paramType = parseTypeSpecifier();
            std::string paramName;
            
            if (check(Lexer::TokenType::IDENTIFIER)) {
                paramName = current().lexeme;
                advance();
            }
            
            func->params.emplace_back(paramType, paramName);
        } while (match(Lexer::TokenType::PUNCT_COMMA));
    }
    
    match(Lexer::TokenType::PUNCT_RPAREN);
    
    // Parse function body
    func->body = parseCompoundStatement();
    
    return func;
}

std::string Parser::parseTypeSpecifier() {
    switch (current().type) {
        case Lexer::TokenType::KEYWORD_INT: advance(); return "int";
        case Lexer::TokenType::KEYWORD_FLOAT: advance(); return "float";
        case Lexer::TokenType::KEYWORD_DOUBLE: advance(); return "double";
        case Lexer::TokenType::KEYWORD_CHAR: advance(); return "char";
        case Lexer::TokenType::KEYWORD_VOID: advance(); return "void";
        case Lexer::TokenType::IDENTIFIER: {
            std::string name = current().lexeme;
            advance();
            return name;  // User-defined type
        }
        default:
            error("Expected type specifier");
            return "int";
    }
}

// ============================================================================
// PARSING: STATEMENTS
// ============================================================================

std::unique_ptr<Stmt> Parser::parseStatement() {
    if (check(Lexer::TokenType::PUNCT_LBRACE)) {
        return parseCompoundStatement();
    }
    
    if (check(Lexer::TokenType::KEYWORD_IF)) {
        return parseSelectionStatement();
    }
    
    if (check(Lexer::TokenType::KEYWORD_WHILE) || check(Lexer::TokenType::KEYWORD_FOR)) {
        return parseIterationStatement();
    }
    
    if (check(Lexer::TokenType::KEYWORD_RETURN) ||
        check(Lexer::TokenType::KEYWORD_BREAK) ||
        check(Lexer::TokenType::KEYWORD_CONTINUE)) {
        return parseJumpStatement();
    }
    
    // Expression statement
    auto stmt = std::make_unique<ExpressionStmt>();
    
    if (!check(Lexer::TokenType::PUNCT_SEMICOLON)) {
        stmt->expr = parseExpression();
    }
    
    match(Lexer::TokenType::PUNCT_SEMICOLON);
    return stmt;
}

std::unique_ptr<CompoundStmt> Parser::parseCompoundStatement() {
    auto block = std::make_unique<CompoundStmt>();
    
    match(Lexer::TokenType::PUNCT_LBRACE);
    
    while (!check(Lexer::TokenType::PUNCT_RBRACE) && 
           !check(Lexer::TokenType::TOKEN_EOF)) {
        block->statements.push_back(parseStatement());
    }
    
    match(Lexer::TokenType::PUNCT_RBRACE);
    
    return block;
}

std::unique_ptr<Stmt> Parser::parseSelectionStatement() {
    match(Lexer::TokenType::KEYWORD_IF);
    match(Lexer::TokenType::PUNCT_LPAREN);
    
    auto cond = parseExpression();
    
    match(Lexer::TokenType::PUNCT_RPAREN);
    
    auto thenBranch = parseStatement();
    std::unique_ptr<Stmt> elseBranch = nullptr;
    
    if (match(Lexer::TokenType::KEYWORD_ELSE)) {
        elseBranch = parseStatement();
    }
    
    return std::make_unique<IfStmt>(std::move(cond), std::move(thenBranch), 
                                     std::move(elseBranch));
}

std::unique_ptr<Stmt> Parser::parseIterationStatement() {
    if (check(Lexer::TokenType::KEYWORD_WHILE)) {
        match(Lexer::TokenType::KEYWORD_WHILE);
        match(Lexer::TokenType::PUNCT_LPAREN);
        
        auto cond = parseExpression();
        
        match(Lexer::TokenType::PUNCT_RPAREN);
        
        auto body = parseStatement();
        
        return std::make_unique<WhileStmt>(std::move(cond), std::move(body));
    }
    
    // For loop
    match(Lexer::TokenType::KEYWORD_FOR);
    match(Lexer::TokenType::PUNCT_LPAREN);
    
    std::unique_ptr<Expr> init;
    std::unique_ptr<Expr> cond;
    std::unique_ptr<Expr> incr;
    
    if (!check(Lexer::TokenType::PUNCT_SEMICOLON)) {
        init = parseExpression();
    }
    match(Lexer::TokenType::PUNCT_SEMICOLON);
    
    if (!check(Lexer::TokenType::PUNCT_SEMICOLON)) {
        cond = parseExpression();
    }
    match(Lexer::TokenType::PUNCT_SEMICOLON);
    
    if (!check(Lexer::TokenType::PUNCT_RPAREN)) {
        incr = parseExpression();
    }
    match(Lexer::TokenType::PUNCT_RPAREN);
    
    auto body = parseStatement();
    
    return std::make_unique<ForStmt>(std::move(init), std::move(cond),
                                      std::move(incr), std::move(body));
}

std::unique_ptr<Stmt> Parser::parseJumpStatement() {
    if (check(Lexer::TokenType::KEYWORD_RETURN)) {
        match(Lexer::TokenType::KEYWORD_RETURN);
        
        std::unique_ptr<Expr> expr;
        if (!check(Lexer::TokenType::PUNCT_SEMICOLON)) {
            expr = parseExpression();
        }
        match(Lexer::TokenType::PUNCT_SEMICOLON);
        
        return std::make_unique<ReturnStmt>(std::move(expr));
    }
    
    if (check(Lexer::TokenType::KEYWORD_BREAK)) {
        match(Lexer::TokenType::KEYWORD_BREAK);
        match(Lexer::TokenType::PUNCT_SEMICOLON);
        return std::make_unique<ExpressionStmt>();
    }
    
    match(Lexer::TokenType::KEYWORD_CONTINUE);
    match(Lexer::TokenType::PUNCT_SEMICOLON);
    return std::make_unique<ExpressionStmt>();
}

// ============================================================================
// PARSING: EXPRESSIONS
// ============================================================================

std::unique_ptr<Expr> Parser::parseExpression() {
    return parseAssignment();
}

std::unique_ptr<Expr> Parser::parseAssignment() {
    auto left = parseLogicalOr();
    
    if (check(Lexer::TokenType::OP_ASSIGN)) {
        advance();
        auto right = parseAssignment();
        return std::make_unique<BinaryExpr>(BinaryOp::ASSIGN, 
                                              std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseLogicalOr() {
    auto left = parseLogicalAnd();
    
    while (check(Lexer::TokenType::OP_OR)) {
        advance();
        auto right = parseLogicalAnd();
        left = std::make_unique<BinaryExpr>(BinaryOp::OR, 
                                             std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseLogicalAnd() {
    auto left = parseEquality();
    
    while (check(Lexer::TokenType::OP_AND)) {
        advance();
        auto right = parseEquality();
        left = std::make_unique<BinaryExpr>(BinaryOp::AND, 
                                             std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseEquality() {
    auto left = parseRelational();
    
    while (checkAny({Lexer::TokenType::OP_EQ, Lexer::TokenType::OP_NE})) {
        BinaryOp op = check(Lexer::TokenType::OP_EQ) ? BinaryOp::EQ : BinaryOp::NE;
        advance();
        auto right = parseRelational();
        left = std::make_unique<BinaryExpr>(op, std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseRelational() {
    auto left = parseAdditive();
    
    while (checkAny({Lexer::TokenType::OP_LT, Lexer::TokenType::OP_GT,
                     Lexer::TokenType::OP_LE, Lexer::TokenType::OP_GE})) {
        BinaryOp op;
        switch (current().type) {
            case Lexer::TokenType::OP_LT: op = BinaryOp::LT; break;
            case Lexer::TokenType::OP_GT: op = BinaryOp::GT; break;
            case Lexer::TokenType::OP_LE: op = BinaryOp::LE; break;
            case Lexer::TokenType::OP_GE: op = BinaryOp::GE; break;
            default: op = BinaryOp::LT;
        }
        advance();
        auto right = parseAdditive();
        left = std::make_unique<BinaryExpr>(op, std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseAdditive() {
    auto left = parseMultiplicative();
    
    while (checkAny({Lexer::TokenType::OP_PLUS, Lexer::TokenType::OP_MINUS})) {
        BinaryOp op = check(Lexer::TokenType::OP_PLUS) ? BinaryOp::ADD : BinaryOp::SUB;
        advance();
        auto right = parseMultiplicative();
        left = std::make_unique<BinaryExpr>(op, std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseMultiplicative() {
    auto left = parseUnary();
    
    while (checkAny({Lexer::TokenType::OP_STAR, Lexer::TokenType::OP_SLASH,
                      Lexer::TokenType::OP_PERCENT})) {
        BinaryOp op;
        switch (current().type) {
            case Lexer::TokenType::OP_STAR: op = BinaryOp::MUL; break;
            case Lexer::TokenType::OP_SLASH: op = BinaryOp::DIV; break;
            case Lexer::TokenType::OP_PERCENT: op = BinaryOp::MOD; break;
            default: op = BinaryOp::MUL;
        }
        advance();
        auto right = parseUnary();
        left = std::make_unique<BinaryExpr>(op, std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseUnary() {
    if (check(Lexer::TokenType::OP_PLUS) || check(Lexer::TokenType::OP_MINUS) ||
        check(Lexer::TokenType::OP_NOT)) {
        std::string op = current().lexeme;
        advance();
        auto operand = parseUnary();
        return std::make_unique<UnaryExpr>(op, std::move(operand));
    }
    
    return parsePostfix();
}

std::unique_ptr<Expr> Parser::parsePostfix() {
    auto expr = parsePrimary();
    
    while (true) {
        if (check(Lexer::TokenType::PUNCT_LBRACKET)) {
            advance();
            auto index = parseExpression();
            match(Lexer::TokenType::PUNCT_RBRACKET);
            expr = std::make_unique<BinaryExpr>(BinaryOp::ADD, std::move(expr), std::move(index));
        } else if (check(Lexer::TokenType::PUNCT_LPAREN)) {
            // Function call
            auto call = std::make_unique<CallExpr>(
                static_cast<Identifier*>(expr.get())->name);
            advance();
            
            if (!check(Lexer::TokenType::PUNCT_RPAREN)) {
                do {
                    call->args.push_back(parseExpression());
                } while (match(Lexer::TokenType::PUNCT_COMMA));
            }
            
            match(Lexer::TokenType::PUNCT_RPAREN);
            expr = std::move(call);
        } else {
            break;
        }
    }
    
    return expr;
}

std::unique_ptr<Expr> Parser::parsePrimary() {
    if (check(Lexer::TokenType::INTEGER_LITERAL)) {
        auto lit = std::make_unique<IntegerLiteral>(current().intValue);
        advance();
        return lit;
    }
    
    if (check(Lexer::TokenType::FLOAT_LITERAL)) {
        auto lit = std::make_unique<FloatLiteral>(current().floatValue);
        advance();
        return lit;
    }
    
    if (check(Lexer::TokenType::STRING_LITERAL)) {
        auto lit = std::make_unique<StringLiteral>(current().lexeme);
        advance();
        return lit;
    }
    
    if (check(Lexer::TokenType::IDENTIFIER)) {
        auto ident = std::make_unique<Identifier>(current().lexeme);
        advance();
        return ident;
    }
    
    if (check(Lexer::TokenType::PUNCT_LPAREN)) {
        advance();
        auto expr = parseExpression();
        match(Lexer::TokenType::PUNCT_RPAREN);
        return expr;
    }
    
    error("Expected expression");
    return std::make_unique<IntegerLiteral>(0);
}

        INDENT(indent + 2) << "Condition:\n";
        condition->print(indent + 4);
    }
    if (increment) {

    }
```

## 3.8 Error Handling and Recovery

### 3.8.1 Error Recovery Strategies

```cpp
// Error recovery implementation

class Parser {
    // ...
    
    void synchronize() {
        // Skip tokens until we reach a synchronization point
        advance();
        
        while (!check(Lexer::TokenType::TOKEN_EOF)) {
            // Check for statement-ending tokens
            if (prev().type == Lexer::TokenType::PUNCT_SEMICOLON) {
                return;
            }
            
            // Check for statement-starting keywords
            switch (current().type) {
                case Lexer::TokenType::KEYWORD_IF:
                case Lexer::TokenType::KEYWORD_WHILE:
                case Lexer::TokenType::KEYWORD_FOR:
                case Lexer::TokenType::KEYWORD_RETURN:
                case Lexer::TokenType::PUNCT_LBRACE:
                case Lexer::TokenType::PUNCT_RBRACE:
                    return;
                default:
                    advance();
            }
        }
    }
    
    // Panic mode recovery
    void panicMode() {
        errorCount_++;
        
        // Discard tokens until we find a safe point
        while (!check(Lexer::TokenType::TOKEN_EOF)) {
            if (check(Lexer::TokenType::PUNCT_SEMICOLON)) {
                advance();
                return;
            }
            
            if (check(Lexer::TokenType::PUNCT_RBRACE)) {
                return;
            }
            
            advance();
        }
    }
};
```

```cpp
// Good error messages

void Parser::error(const std::string& message) {
    std::cerr << "Error at line " << current().line 
              << ", column " << current().column << ": " 
              << message << "\n";
    std::cerr << "  " << getCurrentLine() << "\n";
    std::cerr << "  " << std::string(current().column - 1, ' ') << "^\n";
}

std::string Parser::getCurrentLine() {
    // Extract the current line from source
    size_t start = source.rfind('\n', current().column);
    size_t end = source.find('\n', current().column);
    return source.substr(start + 1, end - start);
}
```

## 3.1 Introduction

Lexical analysis (scanning) and syntax analysis (parsing) are the first two phases of compilation. Together, they transform raw source text into a structured representation that subsequent phases can process.

```
Source Code          Tokens              AST
    │                   │                  │
    ▼                   ▼                  ▼
┌────────┐    ┌────────────────┐    ┌─────────────┐
│  Text  │───▶│   Lexer        │───▶│   Parser    │
│        │    │   (Scanner)    │    │             │
└────────┘    └────────────────┘    └─────────────┘
                   │                        │
                   ▼                        ▼
              token stream              syntax tree
```

## 3.2 Lexical Analysis (Scanning)

The lexer (or scanner) reads source code as a stream of characters and groups them into meaningful units called **tokens**.

### 3.2.1 Token Categories

| Category | Examples |
|----------|----------|
| Keywords | `if`, `else`, `while`, `return`, `int`, `float` |
| Identifiers | Variable names, function names |
| Operators | `+`, `-`, `*`, `/`, `==`, `<=`, `&&` |
| Literals | Integer (`42`), Float (`3.14`), String (`"hello"`) |
| Punctuation | `(`, `)`, `{`, `}`, `;`, `,`, `.` |
| Special | End of file, Error tokens |

### 3.2.2 Token Definition

```cpp
// Token.h
#ifndef TOKEN_H
#define TOKEN_H

#include <string>

enum class TokenType {
    // Keywords
    KEYWORD_IF,
    KEYWORD_ELSE,
    KEYWORD_WHILE,
    KEYWORD_RETURN,
    KEYWORD_INT,
    KEYWORD_FLOAT,
    KEYWORD_VOID,
    KEYWORD_FOR,
    KEYWORD_DO,
    KEYWORD_BREAK,
    KEYWORD_CONTINUE,
    
    // Literals
    INTEGER_LITERAL,
    FLOAT_LITERAL,
    STRING_LITERAL,
    IDENTIFIER,
    
    // Operators
    PLUS,           // +
    MINUS,          // -
    STAR,           // *
    SLASH,          // /
    PERCENT,        // %
    ASSIGN,         // =
    EQ,             // ==
    NE,             // !=
    LT,             // <
    GT,             // >
    LE,             // <=
    GE,             // >=
    AND,            // &&
    OR,             // ||
    NOT,            // !
    
    // Punctuation
    LPAREN,         // (
    RPAREN,         // )
    LBRACE,         // {
    RBRACE,         // }
    LBRACKET,       // [
    RBRACKET,       // ]
    COMMA,          // ,
    SEMICOLON,      // ;
    COLON,          // :
    DOT,            // .
    
    // Special
    END_OF_FILE,
    UNKNOWN,
    ERROR
};

struct Token {
    TokenType type;
    std::string lexeme;        // The actual text
    int line;
    int column;
    
    union {
        int intValue;
        double floatValue;
    };
    
    Token(TokenType t, const std::string& l, int ln, int col)
        : type(t), lexeme(l), line(ln), column(col) {}
};

#endif // TOKEN_H
```

```cpp
// Lexer.cpp
#include "Lexer.h"
#include <cctype>
#include <stdexcept>

void Lexer::initializeKeywords() {
    keywords = {
        {"if", TokenType::KEYWORD_IF},
        {"else", TokenType::KEYWORD_ELSE},
        {"while", TokenType::KEYWORD_WHILE},
        {"return", TokenType::KEYWORD_RETURN},
        {"int", TokenType::KEYWORD_INT},
        {"float", TokenType::KEYWORD_FLOAT},
        {"void", TokenType::KEYWORD_VOID},
        {"for", TokenType::KEYWORD_FOR},
        {"do", TokenType::KEYWORD_DO},
        {"break", TokenType::KEYWORD_BREAK},
        {"continue", TokenType::KEYWORD_CONTINUE}
    };
}

char Lexer::currentChar() {
    if (isAtEnd()) return '\0';
    return source[pos];
}

char Lexer::peekChar() {
    return peekCharAhead(1);
}

char Lexer::peekCharAhead(int offset) {
    size_t peekPos = pos + offset;
    if (peekPos >= source.size()) return '\0';
    return source[peekPos];
}

void Lexer::advance() {
    if (!isAtEnd()) {
        if (currentChar() == '\n') {
            line++;
            column = 1;
        } else {
            column++;
        }
        pos++;
    }
}

bool Lexer::isAtEnd() {
    return pos >= source.size();
}

void Lexer::skipWhitespace() {
    while (!isAtEnd()) {
        char c = currentChar();
        if (c == ' ' || c == '\t' || c == '\r' || c == '\n') {
            advance();
        } else if (c == '/') {
            // Check for comment
            if (peekChar() == '/') {
                skipComment();
            } else {
                break;
            }
        } else {
            break;
        }
    }
}

void Lexer::skipComment() {
    // Single-line comment
    if (peekChar() == '/') {
        while (!isAtEnd() && currentChar() != '\n') {
            advance();
        }
    }
    // Multi-line comment (simplified)
    else if (peekChar() == '*') {
        advance(); advance();  // Skip /*
        while (!isAtEnd()) {
            if (currentChar() == '*' && peekChar() == '/') {
                advance(); advance();  // Skip */
                break;
            }
            advance();
        }
    }
}

std::vector<Token> Lexer::tokenize() {
    std::vector<Token> tokens;
    
    while (!isAtEnd()) {
        skipWhitespace();
        if (isAtEnd()) break;
        
        int startLine = line;
        int startColumn = column;
        char c = currentChar();
        
        // Identifier or keyword
        if (std::isalpha(c) || c == '_') {
            Token token = identifier();
            tokens.push_back(token);
            continue;
        }
        
        // Number
        if (std::isdigit(c)) {
            Token token = number();
            tokens.push_back(token);
            continue;
        }
        
        // String literal
        if (c == '"') {
            Token token = stringLiteral();
            tokens.push_back(token);
            continue;
        }
        
        // Two-character operators
        Token token;
        switch (c) {
            case '+':
                advance();
                token = makeToken(TokenType::PLUS, "+");
                break;
            case '-':
                advance();
                token = makeToken(TokenType::MINUS, "-");
                break;
            case '*':
                advance();
                token = makeToken(TokenType::STAR, "*");
                break;
            case '/':
                advance();
                token = makeToken(TokenType::SLASH, "/");
                break;
            case '%':
                advance();
                token = makeToken(TokenType::PERCENT, "%");
                break;
            case '(':
                advance();
                token = makeToken(TokenType::LPAREN, "(");
                break;
            case ')':
                advance();
                token = makeToken(TokenType::RPAREN, ")");
                break;
            case '{':
                advance();
                token = makeToken(TokenType::LBRACE, "{");
                break;
            case '}':
                advance();
                token = makeToken(TokenType::RBRACE, "}");
                break;
            case '[':
                advance();
                token = makeToken(TokenType::LBRACKET, "[");
                break;
            case ']':
                advance();
                token = makeToken(TokenType::RBRACKET, "]");
                break;
            case ',':
                advance();
                token = makeToken(TokenType::COMMA, ",");
                break;
            case ';':
                advance();
                token = makeToken(TokenType::SEMICOLON, ";");
                break;
            case ':':
                advance();
                token = makeToken(TokenType::COLON, ":");
                break;
            case '.':
                advance();
                token = makeToken(TokenType::DOT, ".");
                break;
            case '=':
                advance();
                token = makeChar('=', TokenType::ASSIGN, TokenType::EQ);
                break;
            case '!':
                advance();
                if (currentChar() == '=') {
                    advance();
                    token = makeToken(TokenType::NE, "!=");
                } else {
                    token = makeToken(TokenType::NOT, "!");
                }
                break;
            case '<':
                advance();
                token = makeChar('=', TokenType::LT, TokenType::LE);
                break;
            case '>':
                advance();
                token = makeChar('=', TokenType::GT, TokenType::GE);
                break;
            case '&':
                advance();
                token = makeChar('&', TokenType::UNKNOWN, TokenType::AND);
                break;
            case '|':
                advance();
                token = makeChar('|', TokenType::UNKNOWN, TokenType::OR);
                break;
            default:
                advance();
                token = makeToken(TokenType::UNKNOWN, std::string(1, c));
                break;
        }
        
        token.line = startLine;
        token.column = startColumn;
        tokens.push_back(token);
    }
    
    tokens.push_back(makeToken(TokenType::END_OF_FILE, ""));
    tokens.back().line = line;
    tokens.back().column = column;
    
    return tokens;
}

Token Lexer::identifier() {
    int startLine = line;
    int startColumn = column;
    std::string lexeme;
    
    while (!isAtEnd() && (std::isalnum(currentChar()) || currentChar() == '_')) {
        lexeme += currentChar();
        advance();
    }
    
    TokenType type = TokenType::IDENTIFIER;
    auto it = keywords.find(lexeme);
    if (it != keywords.end()) {
        type = it->second;
    }
    
    return Token(type, lexeme, startLine, startColumn);
}

Token Lexer::number() {
    int startLine = line;
    int startColumn = column;
    std::string lexeme;
    bool isFloat = false;
    
    while (!isAtEnd() && std::isdigit(currentChar())) {
        lexeme += currentChar();
        advance();
    }
    
    if (currentChar() == '.' && std::isdigit(peekChar())) {
        isFloat = true;
        lexeme += currentChar();
        advance();
        
        while (!isAtEnd() && std::isdigit(currentChar())) {
            lexeme += currentChar();
            advance();
        }
    }
    
    // Scientific notation
    if (currentChar() == 'e' || currentChar() == 'E') {
        lexeme += currentChar();
        advance();
        if (currentChar() == '+' || currentChar() == '-') {
            lexeme += currentChar();
            advance();
        }
        while (!isAtEnd() && std::isdigit(currentChar())) {
            lexeme += currentChar();
            advance();
        }
    }
    
    Token token = Token(
        isFloat ? TokenType::FLOAT_LITERAL : TokenType::INTEGER_LITERAL,
        lexeme, startLine, startColumn
    );
    
    if (isFloat) {
        token.floatValue = std::stod(lexeme);
    } else {
        token.intValue = std::stoi(lexeme);
    }
    
    return token;
}

Token Lexer::stringLiteral() {
    int startLine = line;
    int startColumn = column;
    std::string lexeme;
    advance();  // Skip opening quote
    
    while (!isAtEnd() && currentChar() != '"') {
        if (currentChar() == '\\') {
            advance();
            switch (currentChar()) {
                case 'n': lexeme += '\n'; break;
                case 't': lexeme += '\t'; break;
                case 'r': lexeme += '\r'; break;
                case '"': lexeme += '"'; break;
                case '\\': lexeme += '\\'; break;
                case '0': lexeme += '\0'; break;
                default: lexeme += currentChar(); break;
            }
        } else {
            lexeme += currentChar();
        }
        advance();
    }
    
    if (isAtEnd()) {
        return Token(TokenType::ERROR, lexeme, startLine, startColumn);
    }
    
    advance();  // Skip closing quote
    return Token(TokenType::STRING_LITERAL, lexeme, startLine, startColumn);
}

Token Lexer::makeChar(char expected, TokenType single, TokenType double) {
    if (currentChar() == expected) {
        advance();
        return makeToken(double, std::string(1, currentChar()));
    }
    return makeToken(single, std::string(1, source[pos - 1]));
}

Token Lexer::makeToken(TokenType type, const std::string& lexeme) {
    return Token(type, lexeme, line, column);
}
```

## 3.3 Regular Expressions and Finite Automata

Lexical tokens can be formally specified using **regular expressions (regex)**.

### 3.3.1 Regular Expression Basics

| Pattern | Meaning | Example |
|---------|---------|---------|
| `c` | Literal character | `a` matches "a" |
| `.` | Any character | `a.b` matches "acb", "a1b" |
| `*` | Zero or more | `ab*` matches "a", "ab", "abbb" |
| `+` | One or more | `ab+` matches "ab", "abbb" |
| `?` | Optional | `ab?` matches "a", "ab" |
| `[abc]` | Character class | `[0-9]` matches any digit |
| `^` | Start of line | `^main` matches "main()" |
| `$` | End of line | `main$` matches "...main" |
| `()` | Grouping | `(ab)+` matches "ab", "abab" |

### 3.3.2 Token Patterns

```regex
// Integer literal
[0-9]+

// Float literal
[0-9]+\.[0-9]+

// Identifier
[a-zA-Z_][a-zA-Z0-9_]*

// Keywords
if|else|while|return|int|float|void

// Operators
\+\+|--|==|!=|<=|>=|&&|\|\||[+\-*/%=<>!]

// String literal (simplified)
"[^"]*"
```

### 3.3.3 Finite Automata

Regular expressions are implemented using **Finite Automata**:

- **Deterministic Finite Automaton (DFA)**: One transition per input per state
- **Non-deterministic Finite Automaton (NFA)**: Multiple possible transitions

Thompson's construction converts regex to NFA; subset construction converts NFA to DFA.

```
Example: (a|b)*cd

    ┌─────────────────────────────────────────────┐
    │                                             │
    ▼                                             │
┌───────┐  a,b   ┌───────┐  a,b   ┌───────┐  c  ┌───────┐  d  ┌───┐
│ Start │──────▶│   q   │──────▶│   q   │─────▶│   q   │─────▶│ f │
└───────┘        └───────┘        └───────┘      └───────┘     └───┘
     │               │               │
     └───────────────┴───────────────┘
              (loop back)
```

## 3.4 Syntax Analysis (Parsing)

Parsing analyzes the token stream according to a grammar and produces a parse tree or AST.

### 3.4.1 Context-Free Grammars

A context-free grammar (CFG) consists of:
- **Terminals**: Token types (lowercase or quoted)
- **Non-terminals**: Syntactic categories (uppercase)
- **Production rules**: `A → α`
- **Start symbol**: The entry non-terminal

**Example Grammar for Simple Expressions:**

```ebnf
program     → declaration* 
declaration → type IDENTIFIER ( param_list ) compound_stmt
type        → INT | FLOAT | VOID
param_list  → param ( , param )* | ε
param       → type IDENTIFIER
compound_stmt → { local_declarations* statement* }
statement   → expression_stmt | compound_stmt | selection_stmt | iteration_stmt | return_stmt
expression_stmt → expression? ;
selection_stmt  → IF ( expression ) statement ( ELSE statement )?
iteration_stmt  → WHILE ( expression ) statement
return_stmt     → RETURN expression? ;
expression      → var = expression | simple_expression
var            → IDENTIFIER
simple_expression → additive_expression ( rel_op additive_expression )*
additive_expression → term ( add_op term )*
term          → factor ( mul_op factor )*
factor        → ( expression ) | var | call | NUM | REAL
call          → IDENTIFIER ( arg_list )
arg_list      → expression ( , expression )* | ε
rel_op        → <= | < | > | >= | == | !=
add_op        → + | -
mul_op        → * | /
```

### 3.4.2 Ambiguous Grammars

Grammars that allow multiple parse trees for the same string are ambiguous. The "dangling else" is a classic example:

```
statement → IF expr THEN statement
          | IF expr THEN statement ELSE statement
          | other_stmt

if (x)
    if (y)
        a = 1;
else
    a = 2;  -- Which 'if' does this belong to?
```

**Resolution**: Bind `else` to the nearest `if` (shift-reduce parsing prefers shift).

### 3.4.3 Grammar for Our Language

```ebnf
program        → declaration* 

declaration    → var-declaration | fun-declaration

var-declaration → type-specifier IDENTIFIER ;

fun-declaration → type-specifier IDENTIFIER ( params ) compound-stmt

type-specifier  → INT | FLOAT | VOID

params          → param-list | VOID

param-list       → param ( , param )*

param           → type-specifier IDENTIFIER

compound-stmt   → { local-declarations* statement* }

local-declarations → var-declaration*

statement       → expression-stmt | compound-stmt | selection-stmt | iteration-stmt | return-stmt

expression-stmt → expression ; | ;

selection-stmt  → IF ( expression ) statement ( ELSE statement )?

iteration-stmt  → WHILE ( expression ) statement

return-stmt     → RETURN ; | RETURN expression ;

expression      → var = expression | simple-expression

var            → IDENTIFIER

simple-expression → additive-expression ( relop additive-expression )*

additive-expression → term ( addop term )*

term            → factor ( mulop factor )*

factor          → ( expression ) | var | call | NUM | REAL

call            → IDENTIFIER ( args )

args            → expression ( , expression )* | ε

relop           → <= | < | > | >= | == | !=

addop           → + | -

mulop           → * | /
```

## 3.5 Parser Implementation

We'll implement a **recursive descent parser**, which directly translates grammar rules into functions.

### 3.5.1 AST Node Classes

```cpp
// AST.h
#ifndef AST_H
#define AST_H

#include <vector>
#include <string>
#include <memory>

enum class OpType {
    // Relational
    LT, GT, LE, GE, EQ, NE,
    // Arithmetic
    ADD, SUB, MUL, DIV, MOD,
    // Logical
    AND, OR, NOT,
    // Assignment
    ASSIGN
};

enum class ExprType {
    INTEGER_LITERAL,
    FLOAT_LITERAL,
    IDENTIFIER,
    BINARY,
    UNARY,
    CALL,
    VAR
};

enum class StmtType {
    EXPRESSION,
    COMPOUND,
    SELECTION,
    ITERATION,
    RETURN
};

enum class DeclType {
    VARIABLE,
    FUNCTION
};

class ASTNode {
public:
    virtual ~ASTNode() = default;
    virtual void print(int indent = 0) const = 0;
};

class Expr : public ASTNode {
public:
    ExprType type;
    Expr(ExprType t) : type(t) {}
};

class IntegerLiteral : public Expr {
public:
    int value;
    IntegerLiteral(int val) : Expr(ExprType::INTEGER_LITERAL), value(val) {}
    void print(int indent = 0) const override;
};

class FloatLiteral : public Expr {
public:
    double value;
    FloatLiteral(double val) : Expr(ExprType::FLOAT_LITERAL), value(val) {}
    void print(int indent = 0) const override;
};

class Identifier : public Expr {
public:
    std::string name;
    Identifier(const std::string& n) : Expr(ExprType::IDENTIFIER), name(n) {}
    void print(int indent = 0) const override;
};

class BinaryExpr : public Expr {
public:
    OpType op;
    std::unique_ptr<Expr> left;
    std::unique_ptr<Expr> right;
    
    BinaryExpr(OpType oper, std::unique_ptr<Expr> l, std::unique_ptr<Expr> r)
        : Expr(ExprType::BINARY), op(oper), left(std::move(l)), right(std::move(r)) {}
    void print(int indent = 0) const override;
};

class CallExpr : public Expr {
public:
    std::string callee;
    std::vector<std::unique_ptr<Expr>> args;
    
    CallExpr(const std::string& name)
        : Expr(ExprType::CALL), callee(name) {}
    void print(int indent = 0) const override;
};

class Stmt : public ASTNode {
public:
    StmtType type;
    Stmt(StmtType t) : type(t) {}
};

class ExpressionStmt : public Stmt {
public:
    std::unique_ptr<Expr> expr;
    
    ExpressionStmt(std::unique_ptr<Expr> e)
        : Stmt(StmtType::EXPRESSION), expr(std::move(e)) {}
    void print(int indent = 0) const override;
};

class CompoundStmt : public Stmt {
public:
    std::vector<std::unique_ptr<ASTNode>> declarations;
    std::vector<std::unique_ptr<Stmt>> statements;
    
    CompoundStmt() : Stmt(StmtType::COMPOUND) {}
    void print(int indent = 0) const override;
};

class SelectionStmt : public Stmt {
public:
    std::unique_ptr<Expr> condition;
    std::unique_ptr<Stmt> thenBranch;
    std::unique_ptr<Stmt> elseBranch;
    
    SelectionStmt(std::unique_ptr<Expr> cond, 
                  std::unique_ptr<Stmt> thenStmt,
                  std::unique_ptr<Stmt> elseStmt = nullptr)
        : Stmt(StmtType::SELECTION), condition(std::move(cond)), 
          thenBranch(std::move(thenStmt)), elseBranch(std::move(elseStmt)) {}
    void print(int indent = 0) const override;
};

class IterationStmt : public Stmt {
public:
    std::unique_ptr<Expr> condition;
    std::unique_ptr<Stmt> body;
    
    IterationStmt(std::unique_ptr<Expr> cond, std::unique_ptr<Stmt> b)
        : Stmt(StmtType::ITERATION), condition(std::move(cond)), body(std::move(b)) {}
    void print(int indent = 0) const override;
};

class ReturnStmt : public Stmt {
public:
    std::unique_ptr<Expr> expr;
    
    ReturnStmt(std::unique_ptr<Expr> e = nullptr)
        : Stmt(StmtType::RETURN), expr(std::move(e)) {}
    void print(int indent = 0) const override;
};

class Decl : public ASTNode {
public:
    DeclType type;
    Decl(DeclType t) : type(t) {}
};

class VarDeclaration : public Decl {
public:
    std::string type;
    std::string name;
    int line;
    
    VarDeclaration(const std::string& t, const std::string& n, int ln)
        : Decl(DeclType::VARIABLE), type(t), name(n), line(ln) {}
    void print(int indent = 0) const override;
};

class FunctionDeclaration : public Decl {
public:
    std::string returnType;
    std::string name;
    std::vector<std::pair<std::string, std::string>> params;  // type, name
    std::unique_ptr<CompoundStmt> body;
    
    FunctionDeclaration(const std::string& retType, const std::string& n)
        : Decl(DeclType::FUNCTION), returnType(retType), name(n) {}
    void print(int indent = 0) const override;
};

class Program : public ASTNode {
public:
    std::vector<std::unique_ptr<Decl>> declarations;
    
    void print(int indent = 0) const override;
};

#endif // AST_H
```

### 3.5.2 Parser Implementation

```cpp
// Parser.h
#ifndef PARSER_H
#define PARSER_H

#include "Lexer.h"
#include "AST.h"
#include <vector>
#include <memory>

class Parser {
public:
    Parser(const std::vector<Token>& tokens) : tokens(tokens), current(0) {}
    
    std::unique_ptr<Program> parse();

private:
    std::vector<Token> tokens;
    size_t current;
    
    Token currentToken();
    Token peekToken(int offset = 0);
    Token advanceToken();
    bool check(TokenType type);
    bool checkAny(const std::vector<TokenType>& types);
    bool match(TokenType type);
    bool matchAny(const std::vector<TokenType>& types);
    Token consume(TokenType type, const std::string& message);
    void synchronize();
    
    // Grammar rules
    std::unique_ptr<Program> parseProgram();
    std::unique_ptr<Decl> parseDeclaration();
    std::unique_ptr<VarDeclaration> parseVarDeclaration();
    std::unique_ptr<FunctionDeclaration> parseFunDeclaration();
    std::unique_ptr<Stmt> parseStatement();
    std::unique_ptr<CompoundStmt> parseCompoundStmt();
    std::unique_ptr<Stmt> parseSelectionStmt();
    std::unique_ptr<Stmt> parseIterationStmt();
    std::unique_ptr<Stmt> parseReturnStmt();
    std::unique_ptr<Stmt> parseExpressionStmt();
    std::unique_ptr<Expr> parseExpression();
```

```cpp
// Parser.cpp
#include "Parser.h"
#include <iostream>
#include <cassert>

Token Parser::currentToken() {
    return tokens[current];
}

Token Parser::peekToken(int offset) {
    size_t idx = current + offset;
    if (idx >= tokens.size()) {
        return tokens.back();
    }
    return tokens[idx];
}

Token Parser::advanceToken() {
    if (!check(TokenType::END_OF_FILE)) {
        current++;
    }
    return tokens[current - 1];
}

bool Parser::check(TokenType type) {
    if (isAtEnd()) return false;
    return currentToken().type == type;
}

bool Parser::checkAny(const std::vector<TokenType>& types) {
    for (TokenType type : types) {
        if (check(type)) return true;
    }
    return false;
}

bool Parser::match(TokenType type) {
    if (check(type)) {
        advanceToken();
        return true;
    }
    return false;
}

bool Parser::matchAny(const std::vector<TokenType>& types) {
    for (TokenType type : types) {
        if (check(type)) {
            advanceToken();
            return true;
        }
    }
    return false;
}

Token Parser::consume(TokenType type, const std::string& message) {
    if (check(type)) return advanceToken();
    printError(message);
    return Token(TokenType::ERROR, "", 0, 0);
}

bool Parser::isAtEnd() {
    return currentToken().type == TokenType::END_OF_FILE;
}

void Parser::printError(const std::string& message) {
    std::cerr << "Parse error at line " << currentToken().line 
              << ", column " << currentToken().column << ": " 
              << message << std::endl;
}

std::unique_ptr<Program> Parser::parse() {
    return parseProgram();
}

std::unique_ptr<Program> Parser::parseProgram() {
    auto program = std::make_unique<Program>();
    
    while (!isAtEnd()) {
        try {
            auto decl = parseDeclaration();
            if (decl) {
                program->declarations.push_back(std::move(decl));
            }
        } catch (...) {
            synchronize();
            if (!isAtEnd()) advanceToken();
        }
    }
    
    return program;
}

std::unique_ptr<Decl> Parser::parseDeclaration() {
    std::string type = parseTypeSpecifier();
    
    if (!check(TokenType::IDENTIFIER)) {
        printError("Expected identifier after type");
        return nullptr;
    }
    
    std::string name = advanceToken().lexeme;
    
    if (match(TokenType::LPAREN)) {
        // Function declaration
        auto func = std::make_unique<FunctionDeclaration>(type, name);
        
        // Parse parameters
        if (!check(TokenType::RPAREN)) {
            func->params = parseParamList();
        }
        consume(TokenType::RPAREN, "Expected ')' after parameters");
        
        // Parse function body
        func->body = parseCompoundStmt();
        
        return func;
    } else if (match(TokenType::SEMICOLON)) {
        // Variable declaration
        return std::make_unique<VarDeclaration>(type, name, currentToken().line);
    }
    
    printError("Expected ';' or '(' after identifier");
    return nullptr;
}

std::unique_ptr<VarDeclaration> Parser::parseVarDeclaration() {
    std::string type = parseTypeSpecifier();
    std::string name = consume(TokenType::IDENTIFIER, "Expected identifier").lexeme;
    consume(TokenType::SEMICOLON, "Expected ';' after variable declaration");
    
    return std::make_unique<VarDeclaration>(type, name, currentToken().line);
}

std::unique_ptr<FunctionDeclaration> Parser::parseFunDeclaration() {
    std::string returnType = parseTypeSpecifier();
    std::string name = consume(TokenType::IDENTIFIER, "Expected function name").lexeme;
    
    consume(TokenType::LPAREN, "Expected '(' after function name");
    
    auto func = std::make_unique<FunctionDeclaration>(returnType, name);
    
    // Parse parameters
    if (!check(TokenType::RPAREN)) {
        func->params = parseParamList();
    }
    consume(TokenType::RPAREN, "Expected ')' after parameters");
    
    func->body = parseCompoundStmt();
    
    return func;
}

std::vector<std::pair<std::string, std::string>> Parser::parseParamList() {
    std::vector<std::pair<std::string, std::string>> params;
    
    do {
        std::string type = parseTypeSpecifier();
        std::string name = consume(TokenType::IDENTIFIER, "Expected parameter name").lexeme;
        params.emplace_back(type, name);
    } while (match(TokenType::COMMA));
    
    return params;
}

std::string Parser::parseTypeSpecifier() {
    if (match(TokenType::KEYWORD_INT)) return "int";
    if (match(TokenType::KEYWORD_FLOAT)) return "float";
    if (match(TokenType::KEYWORD_VOID)) return "void";
    
    printError("Expected type specifier");
    return "int";  // Default fallback
}

std::unique_ptr<Stmt> Parser::parseStatement() {
    if (match(TokenType::KEYWORD_IF)) return parseSelectionStmt();
    if (match(TokenType::KEYWORD_WHILE)) return parseIterationStmt();
    if (match(TokenType::KEYWORD_RETURN)) return parseReturnStmt();
    if (match(TokenType::LBRACE)) {
        auto stmt = parseCompoundStmtInternal();
        consume(TokenType::RBRACE, "Expected '}' after compound statement");
        return stmt;
    }
    
    return parseExpressionStmt();
}

std::unique_ptr<CompoundStmt> Parser::parseCompoundStmt() {
    consume(TokenType::LBRACE, "Expected '{'");
    auto stmt = parseCompoundStmtInternal();
    consume(TokenType::RBRACE, "Expected '}' after compound statement");
    return stmt;
}

std::unique_ptr<CompoundStmt> Parser::parseCompoundStmtInternal() {
    auto stmt = std::make_unique<CompoundStmt>();
    
    while (!check(TokenType::RBRACE) && !isAtEnd()) {
        // Check for variable declarations
        if (checkAny({TokenType::KEYWORD_INT, TokenType::KEYWORD_FLOAT, TokenType::KEYWORD_VOID})) {
            if (peekToken(1).type == TokenType::IDENTIFIER && 
                peekToken(2).type == TokenType::SEMICOLON) {
                stmt->declarations.push_back(parseVarDeclaration());
                continue;
            }
        }
        
        // Statement
        auto s = parseStatement();
        if (s) {
            stmt->statements.push_back(std::move(s));
        }
    }
    
    return stmt;
}

std::unique_ptr<Stmt> Parser::parseSelectionStmt() {
    consume(TokenType::LPAREN, "Expected '(' after 'if'");
    auto cond = parseExpression();
    consume(TokenType::RPAREN, "Expected ')' after condition");
    
    auto thenBranch = parseStatement();
    std::unique_ptr<Stmt> elseBranch = nullptr;
    
    if (match(TokenType::KEYWORD_ELSE)) {
        elseBranch = parseStatement();
    }
    
    return std::make_unique<SelectionStmt>(
        std::move(cond), 
        std::move(thenBranch), 
        std::move(elseBranch)
    );
}

std::unique_ptr<Stmt> Parser::parseIterationStmt() {
    consume(TokenType::LPAREN, "Expected '(' after 'while'");
    auto cond = parseExpression();
    consume(TokenType::RPAREN, "Expected ')' after condition");
    
    auto body = parseStatement();
    
    return std::make_unique<IterationStmt>(std::move(cond), std::move(body));
}

std::unique_ptr<Stmt> Parser::parseReturnStmt() {
    std::unique_ptr<Expr> expr = nullptr;
    
    if (!check(TokenType::SEMICOLON)) {
        expr = parseExpression();
    }
    
    consume(TokenType::SEMICOLON, "Expected ';' after return value");
    
    return std::make_unique<ReturnStmt>(std::move(expr));
}

std::unique_ptr<Stmt> Parser::parseExpressionStmt() {
    std::unique_ptr<Expr> expr = nullptr;
    
    if (!check(TokenType::SEMICOLON)) {
        expr = parseExpression();
    }
    
    consume(TokenType::SEMICOLON, "Expected ';' after expression");
    
    return std::make_unique<ExpressionStmt>(std::move(expr));
}

std::unique_ptr<Expr> Parser::parseExpression() {
    return parseAssignment();
}

std::unique_ptr<Expr> Parser::parseAssignment() {
    auto expr = parseSimpleExpression();
    
    if (match(TokenType::ASSIGN)) {
        auto value = parseAssignment();
        return std::make_unique<BinaryExpr>(OpType::ASSIGN, 
            std::move(expr), std::move(value));
    }
    
    return expr;
}

std::unique_ptr<Expr> Parser::parseSimpleExpression() {
    auto left = parseAdditiveExpression();
    
    OpType op = parseRelOp();
    if (op != OpType::ADD) {  // If we got a relational op
        auto right = parseAdditiveExpression();
        return std::make_unique<BinaryExpr>(op, std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseAdditiveExpression() {
    auto left = parseTerm();
    
    while (true) {
        OpType op = parseAddOp();
        if (op != OpType::ADD) break;  // Not an addop
        
        auto right = parseTerm();
        left = std::make_unique<BinaryExpr>(OpType::ADD, std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseTerm() {
    auto left = parseFactor();
    
    while (true) {
        OpType op = parseMulOp();
        if (op != OpType::MUL) break;  // Not a mulop
        
        auto right = parseFactor();
        left = std::make_unique<BinaryExpr>(OpType::MUL, std::move(left), std::move(right));
    }
    
    return left;
}

std::unique_ptr<Expr> Parser::parseFactor() {
    if (match(TokenType::LPAREN)) {
        auto expr = parseExpression();
        consume(TokenType::RPAREN, "Expected ')' after expression");
        return expr;
    }
    
    if (match(TokenType::MINUS)) {
        auto operand = parseFactor();
        return std::make_unique<BinaryExpr>(OpType::SUB, 
            std::make_unique<IntegerLiteral>(0),
            std::move(operand));
    }
    
    if (match(TokenType::KEYWORD_NOT)) {
        // Handle NOT operator (implementation depends on requirements)
        auto operand = parseFactor();
        return operand;  // Simplified - actual implementation would create UnaryExpr
    }
    
    if (match(TokenType::INTEGER_LITERAL)) {
        int val = std::stoi(tokens[current - 1].lexeme);
        return std::make_unique<IntegerLiteral>(val);
    }
    
    if (match(TokenType::FLOAT_LITERAL)) {
        double val = std::stod(tokens[current - 1].lexeme);
        return std::make_unique<FloatLiteral>(val);
    }
    
    if (match(TokenType::IDENTIFIER)) {
        std::string name = tokens[current - 1].lexeme;
        
        if (match(TokenType::LPAREN)) {
            auto call = std::make_unique<CallExpr>(name);
            
            if (!check(TokenType::RPAREN)) {
                do {
                    call->args.push_back(parseExpression());
                } while (match(TokenType::COMMA));
            }
            
            consume(TokenType::RPAREN, "Expected ')' after arguments");
            return call;
        }
        
        return std::make_unique<Identifier>(name);
    }
    
    printError("Unexpected token in expression");
    advanceToken();
    return std::make_unique<IntegerLiteral>(0);
}

OpType Parser::parseRelOp() {
    if (match(TokenType::LE)) return OpType::LE;
    if (match(TokenType::LT)) return OpType::LT;
    if (match(TokenType::GT)) return OpType::GT;
    if (match(TokenType::GE)) return OpType::GE;
    if (match(TokenType::EQ)) return OpType::EQ;
    if (match(TokenType::NE)) return OpType::NE;
    return OpType::ADD;  // Sentinel value
}

OpType Parser::parseAddOp() {
    if (match(TokenType::PLUS)) return OpType::ADD;
    if (match(TokenType::MINUS)) return OpType::SUB;
    return OpType::ADD;  // Sentinel value
}

OpType Parser::parseMulOp() {
    if (match(TokenType::STAR)) return OpType::MUL;
    if (match(TokenType::SLASH)) return OpType::DIV;
    if (match(TokenType::PERCENT)) return OpType::MOD;
    return OpType::MUL;  // Sentinel value
}

void Parser::synchronize() {
    advanceToken();
    
    while (!isAtEnd()) {
        if (tokens[current - 1].type == TokenType::SEMICOLON) {
            return;
        }
        
        switch (currentToken().type) {
            case TokenType::KEYWORD_IF:
            case TokenType::KEYWORD_WHILE:
            case TokenType::KEYWORD_RETURN:
            case TokenType::KEYWORD_INT:
            case TokenType::KEYWORD_FLOAT:
            case TokenType::KEYWORD_VOID:
                return;
            default:
                advanceToken();
        }
    }
}
```

```cpp
void BinaryExpr::print(int indentLevel) const {

    const char* opStr;
    switch (op) {
        case OpType::ADD: opStr = "+"; break;
        case OpType::SUB: opStr = "-"; break;
        case OpType::MUL: opStr = "*"; break;
        case OpType::DIV: opStr = "/"; break;
        case OpType::MOD: opStr = "%"; break;
        case OpType::LT: opStr = "<"; break;
        case OpType::GT: opStr = ">"; break;
        case OpType::LE: opStr = "<="; break;
        case OpType::GE: opStr = ">="; break;
        case OpType::EQ: opStr = "=="; break;
        case OpType::NE: opStr = "!="; break;

    left->print(indentLevel + 2);
    right->print(indentLevel + 2);
```

## 3.6 Using Clang's Parser Library

Rather than building your own lexer and parser, you can use Clang's libTooling for production-quality parsing.

### 3.6.1 AST Dump Example

```cpp
// ast_dump.cpp - Using Clang to dump AST
#include <clang/AST/ASTConsumer.h>
#include <clang/AST/RecursiveASTVisitor.h>
#include <clang/Frontend/CompilerInstance.h>
#include <clang/Frontend/FrontendAction.h>
#include <clang/Tooling/Tooling.h>
#include <iostream>

using namespace clang;

class ASTPrinter : public RecursiveASTVisitor<ASTPrinter> {
public:
    bool VisitFunctionDecl(FunctionDecl *decl) {
        std::cout << "Function: " << decl->getNameAsString() << std::endl;
        return true;
    }
    
    bool VisitBinaryOperator(BinaryOperator *op) {
        std::cout << "  BinaryOp: " << op->getOpcodeStr().str() << std::endl;
        return true;
    }
};

class ASTConsumer : public ASTConsumer {
public:
    void HandleTranslationUnit(ASTContext &context) override {
        ASTPrinter printer;
        printer.TraverseDecl(context.getTranslationUnitDecl());
    }
};

class PrintASTAction : public ASTFrontendAction {
public:
    std::unique_ptr<ASTConsumer> CreateASTConsumer(
        CompilerInstance &CI, StringRef InFile) override {
        return std::make_unique<ASTConsumer>();
    }
};

int main(int argc, char **argv) {
    if (argc < 2) {
        std::cerr << "Usage: " << argv[0] << " <source-file>" << std::endl;
        return 1;
    }
    
    clang::tooling::runToolOnCode(std::make_unique<PrintASTAction>(), argv[1]);
}
```

## 3.7 Parser Generators

While hand-written parsers give you full control, parser generators can speed up development.

### 3.7.1 LLVM's TableGen

LLVM uses TableGen for declarative target description files, but it can also be used for other purposes.

### 3.7.2 Clang's Recursive Descent Parser

Clang uses a hand-written recursive descent parser for C-family languages, demonstrating that for complex languages, hand-written parsers often outperform generated ones.

### 3.7.3 LLVM IR Parser

LLVM provides its own IR parser:
